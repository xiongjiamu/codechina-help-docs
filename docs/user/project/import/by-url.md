# 通过URL导入[](#通过URL导入 "Permalink")

您可以通过提供 Git URL 来导入现有的代码仓库：

1.  在您的仪表板中，单击" **新建项目"**
2.  切换到**导入项目**选项卡
3.  点击**按 URL Repo**按钮
4.  填写" Git 代码仓库 URL"和其余项目字段
5.  单击**创建项目**开始导入过程
6.  完成后，您将被重定向到新创建的项目
